# ND-net
This GitHub repository is associated with our work: Progressive dual-branch adversarial diffusion for sparse-view photoacoustic tomography reconstruction.

## Introdution
This study proposed ND-net, a progressive double adversarial diffusion framework for efficient and high-quality sparse-view PACT reconstruction.

## Open Source
The paper has been submitted to Medical Image Analysis. Once the article is published, we will open the source code of the project.
